# IDP CAMP 2026

ระบบจัดการค่าย IDP CAMP 2026

## การใช้งาน
- `index.html` — QR Code Generator
- `camper.html` — หน้าน้องค่าย
- `staff.html` — หน้าพี่สต๊าฟ (PIN: 1111, 2222)
- `admin.html` — หน้า Admin (PIN: 9999)
- `live.html` — Live Leaderboard สำหรับโปรเจกเตอร์

## Tech Stack
- Firebase Realtime Database (sync real-time)
- Cloudinary (อัปโหลดรูป/วิดีโอ)
- GitHub Pages (hosting)
